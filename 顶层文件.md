项目结构如下，按功能域分组：

------

## 顶层文件

| 文件                 | 用途                                                         |
| -------------------- | ------------------------------------------------------------ |
| `bot.py`             | **主入口**。启动 QQ 机器人，含 EULA 确认、信号处理、worker/runner 进程模型 |
| `saka.py`            | 命令行交互入口，BufferCLI 模式，不连 QQ 也能在终端聊         |
| `Start-ProjectChie.bat`   | Windows 一键启动脚本，调 `.venv\Scripts\python.exe bot.py`   |
| `pyproject.toml`     | Python 项目元数据、依赖声明、ruff 配置                       |
| `requirements.txt`   | pip 安装依赖清单（由 pyproject.toml 生成）                   |
| `uv.lock`            | uv 包管理器锁文件                                            |
| `Dockerfile`         | Docker 镜像构建（Python 3.13-slim + uv）                     |
| `docker-compose.yml` | 编排 core + NapCat（QQ 桥）+ sqlite-web + 可选 Caddy         |

| `locales/`       | i18n 翻译文件，同样按语言分，用于 WebUI 和日志的国际化       |
| `scripts/`       | 工具脚本：启动脚本、评估脚本、i18n 提取/校验、记忆检索测试等 |
| `docs/`          | 开发者文档（贡献指南、i18n 文档等）                          |
| `changelogs/`    | 各版本 changelog                                             |
| `pytests/`       | Python 测试套件                                              |
| `tests/`         | 额外测试目录                                                 |
| `depends-data/`  | 静态资源（角色立绘、默认图片等）                             |
| `logs/`          | 运行日志（LLM 请求日志、Maisaka prompt 日志等）              |
| `temp/`          | 临时文件                                                     |
| `code_scripts/`  | 辅助代码脚本                                                 |
| `.github/`       | CI/CD workflow、Issue 模板                                   |
| `.devcontainer/` | VS Code Dev Container 配置                                   |

## `src/` 核心模块（按功能）

### 主循环与推理

| 模块              | 用途                                                         |
| ----------------- | ------------------------------------------------------------ |
| `src/maisaka/`    | **核心对话引擎**。推理循环、规划器（Planner）、工具提供、上下文消息构建、视觉刷新。子目录：`builtin_tool/`（内置工具）、`display/`（展示格式化）、`reply_effect/`（回复效果评分） |
| `src/chat/`       | **聊天管理**。消息收发（`chat_manager`、`chat_bot`）、回复生成（`replyer/`）、知识图谱（`knowledge/`）、心情流（`heart_flow/`）、图片系统（`image_system/`） |
| `src/llm_models/` | **LLM 调用层**。OpenAI 兼容 SDK 封装、请求快照、异常处理。`model_client/` 是协议适配，`payload_content/` 是多模态载荷处理 |

### 记忆与人格

| 模块                | 用途                                                         |
| ------------------- | ------------------------------------------------------------ |
| `src/A_memorix/`    | **长期记忆子系统 v2**。向量检索 + 图关系检索 + 情景/人物/知识记忆。有自己的 `core/` 引擎和 `scripts/` 工具。修改策略独立 |
| `src/person_info/`  | **用户画像系统**。长期跟踪群友言行，基于人格理论建立心理档案 |
| `src/learners/`     | **学习模块**。表情学习器（自动收藏表情包）、黑话挖掘/解释（识别群内自创词汇） |
| `src/emoji_system/` | **表情管理系统**。定期维护表情缓存、LLM 工具集成             |

### 平台与插件

| 模块                  | 用途                                                         |
| --------------------- | ------------------------------------------------------------ |
| `src/plugin_runtime/` | **插件运行时**。依赖管线、Hook 系统、能力注册（`capabilities/`）、消息转发（`host/`）、与子进程插件通信（`transport/`） |
| `src/platform_io/`    | **平台抽象层**。消息去重、路由、出站追踪、平台注册表。`drivers/` 是具体平台适配 |
| `src/mcp_module/`     | **MCP 协议集成**。连接管理、Hook 桥接、LLM 桥、stdio 过滤器  |
| `src/plugins/`        | 内置插件（如 `built_in/plugin_management` 插件管理）         |

### 通信与 WebUI

| 模块                 | 用途                                                         |
| -------------------- | ------------------------------------------------------------ |
| `src/webui/`         | **Web 管理台后端**。FastAPI REST + WebSocket 端点。`routers/` 是路由（聊天、配置、WS）、`middleware/` 是中间件、`schemas/` 是数据校验 |
| `src/local_console/` | **本地私密控制台**。独立 FastAPI 服务，JSON 会话存储，Ollama 直连，TTS 集成，`static/index.html` 是单页前端 |
| `src/cli/`           | **命令行界面**。BufferCLI 终端交互，控制台格式化             |

### 服务与基础设施

| 模块            | 用途                                                         |
| --------------- | ------------------------------------------------------------ |
| `src/services/` | **服务层**。LLM 服务、嵌入服务、记忆服务、消息发送服务、HTML 渲染、统计服务、数据库服务 |
| `src/config/`   | **配置管理**。TOML 配置加载/热重载、模型配置、Legacy 迁移、官方默认配置 |
| `src/common/`   | **公共库**。日志（structlog）、i18n、数据库（SQLAlchemy/SQLModel + FAISS）、消息仓库、数据模型（`data_models/`）、工具函数（`utils/` 含 TTS）、数据库迁移（`database/migrations/`） |
| `src/core/`     | **核心抽象**。事件总线、类型定义、广播管理器                 |
| `src/manager/`  | **任务管理**。异步任务调度、本地存储管理                     |
| `src/prompt/`   | **提示词管理**。加载和组装 LLM 提示词模板                    |
